CONFLICT RESOLUTION
───────────────────────────────────────────────────────────────────────

IF BOTH AGENTS TRY TO MODIFY SAME FILE:

  1. Check logs/coordination.log timestamps
  2. Agent with earlier timestamp wins
  3. Later agent pulls changes and re-applies
  4. Both document in logs/conflicts/YYYYMMDD.log

IF TESTS FAIL AFTER INTEGRATION:

  1. Codex 1 checks backend tests
  2. Codex 2 checks frontend tests
  3. Both coordinate via logs/test-failures.log
  4. Fix in sequence, not parallel

IF DEPLOYMENT FAILS:

  1. Codex 2 owns deployment
  2. Can request Codex 1 assistance via logs/deployment-help.log
  3. Codex 1 responds with backend fixes if needed
