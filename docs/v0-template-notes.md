## Validation Schema (Zod)

```typescript
const postSchema = z.object({
  title: z.string().min(10).max(200),
  topic: z.string().min(5),
  kind: z.enum(["article", "email", "social_script", "ad_copy"]),
  tone: z.enum(["professional", "casual", "witty", "formal", "enthusiastic"]),
  audience: z.string().optional(),
  body: z.string().min(100).optional(), // Generated by AI or manual
});
```

**Runtime Validation:** Enforced before submission  
**Type Safety:** Full TypeScript inference from schema
