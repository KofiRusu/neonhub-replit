```
Variance = Budget - Actual Spend
Variance % = ((Budget - Actual) / Budget) √ó 100%
```

- üü¢ **Favorable:** Variance > +5% (under budget)
- üü° **Acceptable:** Variance -5% to +5%
- üî¥ **Unfavorable:** Variance < -5% (over budget)

### 5.3 Root Cause Analysis for Variances

**For any variance >10%, conduct root cause analysis:**

1. **Identify the variance**
   - Which category?
   - How much?
   - When did it occur?

2. **Determine root cause**
   - Estimation error?
   - Scope change?
   - Market rate changes?
   - Unexpected events?

3. **Assess impact**
   - One-time or recurring?
   - Affects other categories?
   - Risk to overall budget?

4. **Develop action plan**
   - Can it be recovered?
   - Need budget reallocation?
   - Require contingency funds?

5. **Prevent recurrence**
   - Update estimates
   - Improve forecasting
   - Add controls
